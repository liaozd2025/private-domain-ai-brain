# 门店诊断规则

## 五大指标定义

| 指标 | 说明 | 默认口径 |
| --- | --- | --- |
| 客流量 | 月均到店总人次，含重复到店 | 可由日均客流量 × 月工作天数估算 |
| 人头数 | 实际每月到店的独立人数 | 反映真实可经营客户池 |
| 体验率 | 到店客户中体验项目的人数占比 | 含免费或付费体验 |
| 成交率 | 体验后成交人数占比 | 诊断时需结合体验效果 |
| 成交均价 | 总成交收入 / 成交人数 | 反映方案设计与升单能力 |

## 默认阈值
- 每个美容师每天标准服务人数：`3`
- 体验率目标：`60%`
- 体验率 `40%-60%`：偏低，需提升
- 体验率 `<40%`：明确异常
- 成交率默认目标：`50%`
- 成交均价默认目标：`6000`
- 若门店模型明显偏低客单，可表达为 `5000-6000` 的改善区间

## 约束指标
- 毛利率和经营利润用于约束建议优先级
- 若业绩上升但利润更差，不算有效改善
- 固定成本高、底薪占比高、产品结构差时，优先提示利润问题

## 缺数追问清单
至少缺以下任一项时，应先追问：
- 员工人数
- 日客流量或月客流量
- 人头数
- 体验率
- 成交率
- 成交均价
- 业绩
- 毛利率
- 固定成本、人工成本、经营利润

## 组合诊断规则
- 客流量低 + 人头数低：优先判断为拓客不足、预约不足、休眠客户未激活
- 客流量高 + 体验率低：优先判断为接待过载、客户结构差、员工无暇销售
- 体验率低 + 成交率低：优先检查体验效果、流程执行、报价和赠品机制
- 体验率高 + 成交率低：优先检查体验效果质量、报价策略、销售动作
- 成交率正常 + 成交均价低：优先检查方案设计、升单产品、大单占比
- 成交均价达标 + 利润低：优先检查成本结构、底薪占比、毛利率和产品结构
- 五大指标推算业绩与用户给出的实际业绩差异明显：提示数据口径可能不一致

## 输出原则
1. 先事实判断，再根因排序
2. 先通用经营建议，再品牌化动作建议
3. 每条建议都要尽量对应具体指标问题
4. 行动计划必须可执行，不能只写空泛口号
