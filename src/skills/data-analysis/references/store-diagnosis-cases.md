# 门店诊断案例模式

## 高频问题模式

### 1. 客流量不足
常见证据：
- 日客流量低于 `美容师人数 × 3`
- 月客流量明显低于门店承载能力
- 人头数少、休眠客户多、预约不到位

通用动作：
- 做老客激活和预约体系
- 区分目标客、沉睡客、低消费客
- 修复拓客渠道和到店承接流程

品牌化动作：
- 老客裂变卡
- 预约工具
- 团购体验卡

### 2. 客流量过载但利润差
常见证据：
- 客流量高，但体验率、成交率、利润低
- 员工忙于服务，没时间做销售动作
- 客户结构差，无效客流过多

通用动作：
- 控制无效客流，筛选目标客
- 调整预约密度和服务节奏
- 把销售时间从服务中释放出来

品牌化动作：
- 用预约工具重排到店节奏
- 用活动和分销机制筛选更高价值客流

### 3. 体验率低
常见证据：
- 体验率明显低于 `60%`
- 员工没有固定目标客和体验动作
- 产品/服务效果不明显，体验无法打动客户

通用动作：
- 给每位美容师设体验目标
- 标准化体验流程、效果留证、复盘机制
- 优先把目标客推进到体验而不是停留在常规服务

品牌化动作：
- 套盒体验
- 体验奖励和效果打卡

### 4. 成交率低
常见证据：
- 体验人数不低，但成交人数偏少
- 报价体系混乱，赠品设计弱
- 顾问/店长/美容师职责混淆

通用动作：
- 校正报价梯度和方案设计
- 强化体验效果和成交流程
- 拆清顾问、店长、美容师的销售职责

品牌化动作：
- 刷卡后多刷奖励
- 套盒组合上浮

### 5. 成交均价低
常见证据：
- 成交率还可以，但客单拉不上去
- 方案单一，小单多，大单少
- 没有升单产品和大单设计

通用动作：
- 补足升单产品和年度/季度方案
- 区分小单多刷和大单拉均两条线
- 建立报价、赠品、升单的标准动作

品牌化动作：
- 多刷送产品
- 大单组合包

### 6. 利润低
常见证据：
- 毛利率不低，但经营利润低
- 固定成本高，底薪占比高
- 产品损耗或低毛利项目占比高

通用动作：
- 先重构成本和人效，再追业绩
- 优化低效岗位和人员结构
- 提高高毛利项目占比，控制损耗

品牌化动作：
- 用分销/股东机制降低纯投放获客压力

## 组织问题模式
- 店长不抓销售：要补店长销售职责、见诊职责、考核目标
- 顾问兼操作：销售动作会断裂，应明确销售角色边界
- 美容师只服务不销售：体验率和成交率通常会掉
- 缺考核机制：常见表现是体验率低、流程执行散、员工挑客

## 行动计划表写法
- 项目：只写一个可执行主题，如“提升体验率”
- 当前现状/差额：写清当前值与目标差额
- 目标标准：优先使用五大指标默认阈值或用户自定义标准
- 核心动作：写清具体动作，不写空话
- 机制/话术：写激励、追踪、报价、预约等机制
- 责任人：优先落到老板、店长、顾问、美容师
- 预期目标：写清期望变化，如体验率从 30% 提升到 60%
