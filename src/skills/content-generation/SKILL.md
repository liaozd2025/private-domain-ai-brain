# 内容生成技能规范 (SKILL.md)

## 支持内容类型

| 类型 | 平台 | 长度 | 核心要求 |
|------|------|------|--------|
| 朋友圈文案 | 微信朋友圈 | ≤150字 | 钩子+价值+CTA |
| 小红书笔记 | 小红书 | 500-1000字 | 种草感+干货+标签 |
| 活动方案 | 通用 | 800-2000字 | 完整策划框架 |
| SOP文档 | 内部文档 | 500-3000字 | 步骤清晰可执行 |
| 销售话术 | 私聊/群聊 | 300-800字 | 自然不硬核 |
| 企微推文 | 企微群/私聊 | 200-500字 | 亲切有行动导向 |
| 抖音脚本 | 抖音 | 60-120s | 前3秒钩子 |
| 社群公告 | 微信群 | 100-300字 | 简洁清晰 |

## 生成规范

### 禁用词汇（广告法）
- 最佳、最优、第一、唯一 → 改为"优质"、"专业"、"领先"
- 100%有效、绝对 → 改为"大多数客户反馈"、"通常"
- 国家级、全球级 → 需要资质证明

### 不同渠道的语气差异

**企微私聊/群聊**：
```
亲，您好！[开门见山]
今天想跟您分享[主题]...
如有疑问随时@我 😊
```

**朋友圈**：
```
[强钩子开头]
[核心价值，简洁]
点击链接/联系我：[CTA]
```

**小红书**：
```
[数字化标题] 例：3步搞定XXX！
[个人经验分享感]
📌 干货要点：
#私域运营 #话题标签
```
