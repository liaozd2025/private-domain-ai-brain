# 私域运营领域知识 (SKILL.md)

## 核心概念词汇表

| 术语 | 解释 |
|------|------|
| 私域流量 | 企业自主拥有的用户资产（微信好友、社群成员、会员等） |
| 公域流量 | 平台上的开放流量（抖音推荐、小红书搜索等） |
| 裂变 | 通过老用户带新用户的增长方式 |
| 复购率 | 在一定时间内重复购买的用户比例 |
| LTV | 用户生命周期价值（Life Time Value） |
| CAC | 用户获取成本（Customer Acquisition Cost） |
| 会员分层 | 按消费频次/金额将用户分级运营 |
| 企微 | 企业微信，私域运营核心工具 |
| SCRM | 社会化客户关系管理系统 |
| 社群 | 微信群/企微群等社交社区 |

## 私域运营核心指标

| 指标 | 说明 | 行业基准 |
|------|------|---------|
| 入群转化率 | 触达用户中加入社群的比例 | 30-60% |
| 群活跃率 | 每日/周有互动的群成员比例 | 10-30% |
| 复购率 | 3个月内二次购买比例 | 20-40% |
| 裂变系数 | 平均每个老用户带来的新用户数 | 1.1-2.0 |
| 月均 GMV/群 | 社群月均产生的销售额 | 视品类不同 |

## 典型私域运营路径

### B2C 门店私域路径
```
线下/线上接触 → 加企微好友 → 拉入品牌群 → 内容种草 → 活动促销 → 复购 → 转介绍
```

### 企微运营关键动作
1. **建联**：收银/导购引导加企微（话术："加企微，享专属优惠"）
2. **分层**：按消费金额/频次打标签分类
3. **激活**：朋友圈+群发+1v1跟进
4. **留存**：定期活动+有价值内容+情感维系
5. **变现**：专属活动、秒杀、拼团

## 私域增长公式

```
私域 GMV = 用户数 × 活跃率 × 转化率 × 客单价 × 购买频次
```

提升路径：
- 用户数：公域引流、老带新裂变
- 活跃率：内容运营、活动策划
- 转化率：精准推荐、限时优惠
- 客单价：组合销售、会员特权
- 购买频次：定期触达、场景培育

## 常见场景最佳实践

### 社群冷启动
1. 种子用户先行（10-30位忠实顾客）
2. 群名/群头像/群公告设计好再拉人
3. 前 3 天每天互动不少于 5 次
4. 首周必有福利（见面礼/签到红包）

### 朋友圈运营
- 内容比例：干货 40% + 生活 30% + 产品 20% + 互动 10%
- 发圈频率：1-3 次/天，避开 9:00-10:00 和 22:00 后
- 配图：真实、有温度，避免过度设计感

### 社群促活
- **早安签到**：每天固定时间，积分/福利激励
- **话题讨论**：发布与用户相关的话题，引导互动
- **限时秒杀**：每周 1-2 次，群内专属，制造紧迫感
- **分享有礼**：鼓励用户晒图/分享，强化口碑

### 私聊跟进节奏
- D+1（加好友次日）：发欢迎消息 + 了解需求
- D+7：分享有价值内容（非推销）
- D+30：活动邀请或关怀问候
- 消费后 24h：满意度跟进
